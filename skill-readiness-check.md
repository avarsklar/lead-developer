**The thought it answers:** *"I don't even know where to start."*

## What it does

- You answer **four shrug-friendly questions** — the kind you can answer even if you're not sure.
- It **reads your app's code itself**, so you don't have to.
- It hands you back two things: **which gate you're at**, and **the single next move** (never more than three).

## When to run it

First — before any other skill. And any time you're unsure where you stand, or you're about to push for more users.

## When to stop or skip

It **only points; it never fixes anything.** That's on purpose — it's a calm diagnosis, not a repair. Re-run it later any time you want to see how far you've climbed.

## Before this, you need

Nothing. This is where everyone starts.

## After this — do this skill

Whatever gate it sends you to. Almost everyone starts at Gate 1 with **[`/safety-net`](skill-safety-net.html)** — run that next.
