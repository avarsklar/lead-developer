# `/readiness-check`

<p class="lead">The front door — and the home base you come back to. It tells you where you stand, which gate you're at, and your next move. Run it first, then run it again any time you feel lost.</p>

**The thought it answers:** *"I don't even know where to start"* — or, later, *"I'm lost. Where am I again, and what was I supposed to do next?"*

## What it does

- You answer four shrug-friendly questions, the kind you can answer even if you're not sure.
- It reads your app's code itself, so you don't have to.
- It hands you back two things: which gate you're at, and what to do next — usually one move, never more than three.
- Come back confused any time and it re-grounds you — where you are, what's done, the one next thing — and answers plain questions about your app, without changing anything.

## When to run it

First, before any other skill. It takes about five minutes. Then come back to it whenever you feel lost, confused, or disorganized, whenever you're unsure where you stand, or you're about to push for more users — it's the safe place to return to.

## When to stop or skip

It's never worth skipping — five minutes, and it's how every climb starts. Don't expect it to fix anything, though. It only points, and that's on purpose.

## What you need first

Nothing. This is where everyone starts.

## What's next

Whatever gate it sends you to. Almost everyone starts at Gate 1 with **[`/safety-net`](skill-safety-net.html)**; run that next.
